How It Works

  1. User requests your website URL
  2. Route 53 resolves DNS to CloudFront
  3. CloudFront serves cached content from edge locations
  4. If not cached, CloudFront fetches from S3 origin
  5. S3 returns the static files (HTML, CSS, JS, images)

AWS Services Used

Amazon S3

Object storage for hosting static website files

Amazon CloudFront

CDN for global content delivery and HTTPS

AWS Certificate Manager

Free SSL/TLS certificates for HTTPS

Amazon Route 53

DNS service for custom domain names
